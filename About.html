DockerFileGenerator

ver. 0.1

Requirements

  • graphical DockerFile generating by drag'n'drop or double-click
  • UI with JavaFX
  • show or hide outputconsole at Generating-Process (log-Entries or Errors/Warnings)
  • errors and hints if the false element has been chosen by user (see XML-Schema)
  • Specifications and Descriptions as a XML-Document for easy Reconfiguration
  • XML-Schema for correct building DockerFiles
  • predefined and ready-to-use DockerFile-Templates (XML)

Technologies

  • Java8 + JavaFX
  • Git, GitHub - for SourceControlManagement
  • Apache Maven - Build- and Dependency-Management
  • JDOM - for XML-Handling
  • slf4j, jul - for Logging
  • JUnit - Testing
